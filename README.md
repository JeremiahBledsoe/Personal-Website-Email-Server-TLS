# Personal Website & Email Server (TLS)

Built a personal website and self-hosted email server using a Linux VPS, ensuring secure and reliable communication with proper DNS and SSL configurations.

## Tools & Tech
- Linux (Debian VPS)
- NGINX
- DNS (MX, SPF, A)
- SSL (Let’s Encrypt)
- Mail server (Postfix)

## Key Tasks
- Configured NGINX to host a static website
- Secured site and email communication with Let’s Encrypt SSL certificates
- Managed DNS for secure mail delivery and domain trust
- Enabled SPF/MX records to improve email reliability

## Skills Demonstrated
- Linux VPS administration
- Web server configuration
- Email server setup and DNS security
